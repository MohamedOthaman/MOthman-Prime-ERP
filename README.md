# Food Choice ERP

Warehouse, inventory, purchasing, sales, and reporting system for Food Choice operations.

## Local development

Requirements:

- Node.js
- npm

Run locally:

```sh
npm install
npm run dev
```

## Main stack

- Vite
- TypeScript
- React
- Tailwind CSS
- Supabase
- Tauri v2 desktop shell scaffold
- Capacitor mobile shell config
